## Daily — the Draw & Bias

```
   ┌──────────┐ daily FAIR VALUE GAP = also a BREAKER (low-high-low)
   │   FVG    │  price filled + tested it, then rallied
   └────┬─────┘
   daily FVG LOW ═══  ← HTF DRAW anchoring every lower-TF move
   bias: bullish until STH taken → 1–2 day retrace into the gap → continuation
```

See [Fair Value Gaps](../../concepts/fair-value-gaps.md) / [Breaker](../../concepts/breaker.md) / [Daily Bias](../../concepts/daily-bias.md) / [Draw on Liquidity](../../concepts/draw-on-liquidity.md).

## Wednesday — Textbook Power of 3 (AMD)

```
   D  distribution: rally, take the STH, close higher ───▶ objective
        ╱
   ────╱  ← bullish market structure shift
   A  ══   open → consolidate (ACCUMULATION)
        ╲
   M      ╲__ dip into the daily FVG low & bounce (MANIPULATION: sweeps liquidity, sets LOD)
```

1. **Open → consolidate** = **Accumulation**.
2. **Dip into the daily FVG low & bounce** = **Manipulation** (sweeps liquidity, sets the **low of the day**). See [Liquidity](../../concepts/liquidity.md).
3. **Rally, take the STH, close higher** = **Distribution** (bullish **MSS** confirms). See [Market Structure Shift](../../concepts/market-structure-shift.md) / [Power of 3](../../concepts/power-of-3.md).

## Scale-Independence

```
   DAILY:  open → A → M (LOD) → D
             ↓ same story
   5/1-MIN: open → A → M → D   ← "not noise" — the daily story on a smaller interval
```

The 1- and 5-minute chop is the **same AMD story**. See [Power of 3](../../concepts/power-of-3.md) / [Fractals](../../concepts/fractals.md).

## NFP Friday & the 2-min

- **NFP Friday:** don't speculate — **study** which side of liquidity price reaches before/after **8:30**. See [Economic Calendar](../../concepts/economic-calendar.md).
- **2-min:** NY **lunch low** stop run → drift up into the old high; **no clean model entry**, but a **fair value gap** enables the continuation higher. See [Fair Value Gaps](../../concepts/fair-value-gaps.md).

## The Repeatable Read

```
Daily FVG/breaker = draw + bullish bias
→ day unfolds as AMD: open → accumulate → manipulate (LOD at the FVG) → distribute up
→ same story on 1/5-min (not noise) → PM draws into the upside objective
```
