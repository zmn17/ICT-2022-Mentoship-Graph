## Narrative First (HTF Bias & Draw)

```
   ═══════════ relative equal highs  ← DRAW: engineered buy stops (objective)
        ▲
        │  smart money net LONG after purging sell-side
   ─ ─ ─┴─ ─ old lows ═════  sell-side PURGED (stops taken) → net long
```

- Price **purged sell-side liquidity** below old lows → smart money left **net long**. See [Liquidity](../../concepts/liquidity.md).
- The **draw** becomes the **relative equal highs** above (buy stops). See [Draw on Liquidity](../../concepts/draw-on-liquidity.md).

---

## Power of 3 + Judas Swing (building the session low)

```
   midnight open ─┐   8:30 open ─┐
                  │              │
   deep discount ─┴──────────────┴─  price trading BELOW both opens
        ╲  daily Judas swing (pre-NY decline against a bullish day)
         ╲  micro Judas swing (post-8:30 dip) = session manipulation
          ▼ accumulation → manipulation → DISTRIBUTION (up)
```

- Overlay the **midnight** + **8:30** opens → price in **deep discount** (bullish framing). See [Opening Price](../../concepts/opening-price.md).
- Pre-NY decline = **daily Judas swing**; post-8:30 dip = **micro Judas swing**. See [Judas Swing](../../concepts/judas-swing.md).
- **Power of 3** builds both the daily candle and the NY session low. See [Power of 3](../../concepts/power-of-3.md).

---

## Entry (5m/1m: OB + FVG + MSS)

```
   ┌──────────┐  bullish order block (down-close candles)
   │ OB × FVG │  + fair value gap + old swing low = "silver platter" confluence
   └──────────┘  ← BUY in the 62–79% OTE zone after shift in market structure
        │ displacement up qualifies the OB
        ▼ expansion toward the relative equal highs (the draw)
   ═══════════ relative equal highs = objective reached
```

1. On 5m/1m: a **bullish order block + fair value gap + old swing low** line up. See [Order Blocks](../../concepts/order-blocks.md).
2. **Shift in market structure** → entry in the **62–79% OTE** zone. See [Optimal Trade Entry](../../concepts/optimal-trade-entry.md) / [Market Structure Shift](../../concepts/market-structure-shift.md).
3. **Displacement** up qualifies the OB (leaves an FVG). See [Displacement](../../concepts/displacement.md).
4. **Run into the relative equal highs** — the pre-identified objective.

---

## The Repeatable Sequence

```
Purge sell-side → net long → draw = equal highs above
→ deep discount below midnight/8:30 opens → daily + micro Judas swing
→ Power of 3 builds the session low → bullish OB+FVG (state-of-delivery flip)
→ MSS → OTE entry (62–79%) → expand to the equal highs
```

This is the same public "YouTube model" — the three pillars (OB, Judas swing, Power of 3) are one engine.
