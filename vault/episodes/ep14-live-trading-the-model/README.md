## What This Episode Teaches

A short, over-the-shoulder **live execution** of "the model." No new theory — it shows real-time **trade management**: entering long on a limit in a fair value gap after a bullish MSS, targeting a gap fill + buy-side liquidity, scaling out to fund the position, rolling stops to break-even, and cross-checking the correlated S&P.

| Buy-side liquidity | 14,160 | Magnet / run target |
| Initial interest | 14,180 | First objective |
| Session target | 14,220 | Projected objective |
| Exit | near the gap closure | Thesis complete |

---

## Management Lessons

- **Watch the candles**, not the profit number — is price *constantly* working toward the gap?
- **Scale out** to fund the position, then **roll stops to break-even** — risk-free runner toward target
- Correlated **ES** confirmation builds (or removes) confidence
- Exit when the **thesis (gap fill) is nearly complete** — don't overstay
