## The Setup

```
   target 680 FVG ──────────  premium objective (limit @ 80 level, ~50 handles)
        ▲
   ┌────┴────┐ entry FVG  ← BUY on re-entry after bullish MSS
   │  FVG    │
   └────┬────┘
        │  swing high broken = bullish market structure
   ─ ─ ─┴─ ─ ─  swing low
   13,614 ═══════════════  SELL STOPS (expected to be run — tape reading)
```

1. **Daily bias bullish** → longs only. See [Daily Bias](../../concepts/daily-bias.md).
2. **Swing high breaks** → bullish [market structure shift](../../concepts/market-structure-shift.md) confirmed.
3. A **fair value gap** forms; wait for the candle **close** to complete it. See [Fair Value Gaps](../../concepts/fair-value-gaps.md).

---

## Tape Reading the Stops

> Other traders buying this gap rest **sell stops below the swing low at 13,614**. A brief poke below to **run those stops** is *expected* — it **does not negate** the fair value gap because we're trading the **narrative**.

- Buy as price **re-enters the FVG**
- At the **9:30 open**, manage **without a hard stop** (finger on the close button) — resting stops get picked off in the open's whip. See [Tape Reading](../../concepts/tape-reading.md).

---

## Target & Exit

- Upside objective: the **fair value gap at 680** (premium vs equilibrium)
- A small FVG near **630** would support price if revisited
- Placed a **limit at the 80 level** (~50 handles, ~$1,000+); got filled
- **FOMC discipline:** done early — heavy volatility comes later in the day. See [Economic Calendar](../../concepts/economic-calendar.md).

---

## Result

```
2 trades → ~$1,255 net (~$15 commissions)
→ live account equity pushed toward 50,000 → done for the month
```

## The Repeatable Sequence

```
Bullish bias → break swing high (MSS) → FVG forms/closes
→ tape: sell stops @ 13,614 will run → buy FVG re-entry (manage manually)
→ target premium FVG (680) → limit @ 80 → book it → step aside before FOMC
```
