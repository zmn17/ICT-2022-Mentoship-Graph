## What This Teaches

The **discipline filter** for entries, taught through a **failed** setup. The mandatory sequence: (1) price **raids liquidity** (sweeps relative equal highs, or trades into an old FVG), (2) **displacement** — an energetic move that takes out a **short-term low** and **shifts structure** — **must** occur, (3) only *then* return to the resulting **fair value gap** to enter. On this day there was **no displacement**, so **there was no trade**. Anyone who bought impulsively wasn't trading the model.

## The Mandatory Sequence (Entry Filter)

```
1. LIQUIDITY RAID   → sweep relative equal highs (buy stops)
                       OR trade into an old fair value gap
        │  (this ALONE is not a trade — it only sets the stage)
        ▼
2. DISPLACEMENT     → energetic move that takes out a SHORT-TERM LOW
                       and shifts market structure   ← MANDATORY
        │  NO displacement = NO trade
        ▼
3. RETURN TO FVG    → pan back through the displacement leg,
                       enter on the fair value gap it left behind
```

| Step | Action | Concept |
|------|--------|---------|
| 1 | **Time first:** anchor the setup to the **9:30 open** volatility (not random hours). | [Trading Sessions](../../concepts/trading-sessions.md) |
| 2 | **Context:** Thursday pre-NFP, but FOMC already done → low expected impact. | [Economic Calendar](../../concepts/economic-calendar.md) |
| 3 | **Raid:** expect price to **stab above the relative equal highs** (run buy stops). | [Liquidity](../../concepts/liquidity.md) |
| 4 | **Displacement (required):** energetic break of a **short-term low** that shifts structure. | [Displacement](../../concepts/displacement.md) / [MSS](../../concepts/market-structure-shift.md) |
| 5 | **Entry:** return to the **fair value gap** the displacement left. | [Fair Value Gaps](../../concepts/fair-value-gaps.md) |

## Why Today Was a NO-Trade

> Price traded into the shaded area, but the follow-through candle **"gave up the ghost"** — **no displacement** above the old low across the 15m/5m. So the setup was **invalid** and correctly **skipped**. Anyone who bought without displacement was **not** using the model and must **fault themselves** for the loss.

**Skipping non-conforming setups is the discipline being taught.** See [Trading Psychology](../../concepts/trading-psychology.md).

---

## The Mirrored Second Entry

When an **old fair value gap sits above** a bearish market, treat the **low of that gap** exactly like **relative equal highs**:
- Wait for price to trade **up into the gap**
- Then look for the **same displacement-and-return** sequence to short

> Both approaches share **identical logic** — only the **liquidity target** differs (equal highs vs. old FVG).

## Quick-Reference Checklist

```
□ Time anchored to 9:30 (or the relevant kill zone) — time FIRST
□ Macro context checked (NFP/FOMC)
□ Liquidity raided (equal highs swept OR old FVG entered)
□ DISPLACEMENT confirmed (short-term low taken, structure shifted)  ← mandatory
□ Entry on the FVG the displacement left behind
□ No displacement? → NO TRADE (skip it — that IS the edge)
□ Didn't chase / button-push without the sequence
```

The discipline being taught is **skipping** everything that doesn't conform.
