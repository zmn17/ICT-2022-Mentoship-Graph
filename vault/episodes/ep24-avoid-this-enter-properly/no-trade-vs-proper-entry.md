## The Setup Flagged Pre-9:30

- Relative equal highs marked on the **daily & 1-hour** NQ. See [Liquidity](../../concepts/liquidity.md).
- Posted a **discount fair value gap** interest ahead of the **9:30 open** (time first). See [Trading Sessions](../../concepts/trading-sessions.md).
- Context: Thursday pre-**NFP**, but **FOMC already done** → low expected impact. See [Economic Calendar](../../concepts/economic-calendar.md).

---

## Why It Was a NO-Trade

```
   relative equal highs ═══
        ╱  price traded into the shaded area…
   ────╱
        ╲  follow-through candle "gave up the ghost"
         ╳  NO displacement above/below the old low (15m & 5m)
            → structure did NOT shift → SETUP INVALID → skip
```

- Price reached the point of interest but produced **no displacement** taking out a short-term low.
- Without displacement, there is **no shift in structure** and **no fair value gap to enter on** → **no trade**.
- Buying here = impulsive **button-pushing**, not the model. See [Displacement](../../concepts/displacement.md) / [Trading Psychology](../../concepts/trading-psychology.md).

---

## What a PROPER Entry Requires (Bearish)

```
1. sweep relative equal highs (run buy stops)
        ╱╲
   ────╱  ╲
2.        ╲  DISPLACEMENT down → takes out a short-term low → MSS
           ┌──────┐
3.         │ FVG  │ ← return here to SHORT (the gap displacement left)
           └──────┘
                ▼ toward the draw
```

See [Displacement](../../concepts/displacement.md) / [Market Structure Shift](../../concepts/market-structure-shift.md) / [Fair Value Gaps](../../concepts/fair-value-gaps.md).

---

## The Mirrored Entry (Old FVG as Target)

```
   old FVG above ┌──────┐  ← treat the LOW of this gap like relative equal highs
                 │ FVG  │
                 └──────┘
        price trades UP into the gap → then displacement down → return to new FVG → short
```

Same logic — only the **liquidity target** changes (relative equal highs ↔ old FVG low). See [Draw on Liquidity](../../concepts/draw-on-liquidity.md).

---

## The Repeatable Filter

```
liquidity raid  →  DISPLACEMENT (mandatory: short-term low taken, structure shifts)  →  return to FVG = entry
   no displacement  →  NO TRADE (skip — that is the discipline)
```
