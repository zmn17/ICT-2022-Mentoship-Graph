## Daily Context (Higher-Timeframe Bearish)

```
   relative equal highs ═══  DRAW ON LIQUIDITY (magnet UP) — need not be reached
        ▲
        │  week of decline + Friday retrace
        │  counter-trend justified: retrace probable, weekend squeeze
```

Even in a downtrend, the **relative equal highs** above are a **magnet** pulling price up — the counter-trend anchor. See [Draw on Liquidity](../../concepts/draw-on-liquidity.md) / [Daily Bias](../../concepts/daily-bias.md).

---

## Intraday Rebuild (Power of 3)

```
   12:10 lunch high ──╮
        ╱╲            │ retrace into afternoon
   ────╱  ╲──────────╱
   ┌──────┐  ← equilibrium (50%): buy only BELOW
   │ FVG  │  ← DISCOUNT fair value gap = counter-trend LONG
   └──────┘
   9:30 sweeps overnight LONDON LOW = LOW OF DAY (manipulation)
```

- **9:30** sweeps the **overnight London low** → makes the **low of the day** (accumulation/manipulation). See [Power of 3](../../concepts/power-of-3.md).
- Sell-side below equal lows **already taken** → **buy-side** above is the objective; hourly shows a **shift in structure**. See [Liquidity](../../concepts/liquidity.md) / [Market Structure Shift](../../concepts/market-structure-shift.md).

---

## The Entry (Discount + FVG + Displacement)

```
   ── equilibrium ──  buy only below this
   ┌────────┐
   │  FVG   │ ← price retraces into a DISCOUNT fair value gap
   └────────┘
        │ 1:30 algo resumes seeking liquidity
        ▼ displacement OUT of the discount array = bullish MSS → LONG ~2:00
   ▲ price delivered up toward the buy stops (the draw)
```

- Wait for price back into a **discount** (below equilibrium) **and** into a **fair value gap** = buy cheap. See [Market Structure](../../concepts/market-structure.md) / [Fair Value Gaps](../../concepts/fair-value-gaps.md).
- **1:30–2:00**: displacement out of the discount confirms the bullish shift → long. See [Displacement](../../concepts/displacement.md) / [Trading Sessions](../../concepts/trading-sessions.md).
- Refined down through **4/3/2-min** on an **order block + FVG**. See [Order Blocks](../../concepts/order-blocks.md).

---

## The Repeatable Sequence

```
HTF bearish BUT draw (equal highs) is above → up-magnet
→ 9:30 sweeps London low (LOD) → rally → retrace into discount FVG
→ 1:30–2:00 displacement out of discount = bullish MSS → LONG
→ price delivers toward buy stops (draw need not be reached)
```
