## What This Teaches

How to take a **counter-trend intraday long** inside a **higher-timeframe bearish** market — using the **same tools** as trend trades. The anchor is the **draw on liquidity** (relative equal highs above = a magnet). Even in a downtrend, if price is being drawn up, you buy from a **discount** (below equilibrium, into a fair value gap) after a shift in structure. Key nuance: **the draw doesn't have to be reached** to be profitable — you just need the direction to build a bias.

## Learn the Concepts (Read in This Order)

```
1. Draw on Liquidity   → Equal highs above = the magnet (even in a downtrend)
2. Daily Bias          → Counter-trend is evidence-based (long decline + weekend squeeze)
3. Liquidity           → Sell-side below equal lows already taken → buy-side is next
4. Power of 3          → 9:30 sweeps the London low = manipulation (makes LOD)
5. Market Structure (P/D) → Buy from DISCOUNT (below equilibrium)
6. Fair Value Gaps + MSS → Discount FVG + bullish shift = the entry
7. Trading Sessions    → 9:30 / 12:10 lunch / 1:30 seek / 2:00 entry
```

## The Counter-Trend Long (Step-by-Step)

| Step | Action | Concept |
|------|--------|---------|
| 1 | Daily: week of decline, Friday retrace; **relative equal highs above = the draw** (magnet up). | [Draw on Liquidity](../../concepts/draw-on-liquidity.md) |
| 2 | **Counter-trend justification:** long decline → retrace probable; weekend → most want out (squeeze). | [Daily Bias](../../concepts/daily-bias.md) |
| 3 | Hourly: **shift in market structure**; sell-side below equal lows **already taken** → buy-side is the objective. | [Liquidity](../../concepts/liquidity.md) / [MSS](../../concepts/market-structure-shift.md) |
| 4 | **9:30 open** sweeps the **overnight London low** → makes the **low of the day** (Power of 3 accumulation/manipulation). | [Power of 3](../../concepts/power-of-3.md) |
| 5 | Price rallies, makes the high near **12:10 lunch**, then **retraces** into the afternoon. | [Trading Sessions](../../concepts/trading-sessions.md) |

| 7 | **1:30** the algo resumes seeking liquidity; **displacement** out of the discount array confirms the bullish shift → **long ~2:00**. | [Displacement](../../concepts/displacement.md) |
| 8 | Let the algo price higher toward the **buy stops** above. | [Draw on Liquidity](../../concepts/draw-on-liquidity.md) |

## Counter-Trend, Same Tools

```
   equal highs ═══  DRAW (magnet up) — need NOT be reached to profit
        ▲
   ── equilibrium ──  buy only BELOW this (discount)
   ┌────────┐
   │  FVG   │ ← counter-trend LONG (discount + FVG + MSS + displacement)
   └────────┘
   9:30 sweeps London low = LOD (manipulation)
```

> A counter-trend idea is defensible **only** with: a clear **draw on liquidity**, a **shift in market structure**, and an entry **from a discount**. Same FVG/OB principles as a trend trade — just framed against the higher-timeframe bias.

```

```

## The Core Insight

> A **counter-trend** trade uses the **exact same tools** as a trend trade. Anchor on the **draw on liquidity** (equal highs = magnet up, even in a downtrend), wait for price into a **discount** below equilibrium **and** a **fair value gap**, confirm with a **shift in structure/displacement**, and buy cheap. The draw **doesn't have to be reached** — knowing the direction is enough to build the bias.
