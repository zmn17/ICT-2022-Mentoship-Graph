## Reading the Day Type

```
   prior day = OUTSIDE DOWN CLOSE (above prev high & below prev low, closed down)
        → anticipate a CONSOLIDATION day
   levels: hourly high 3933.25 / low 3872.25 / untraded daily low 3855
```

See [Consolidation Day](../../concepts/consolidation-day.md).

---

## Falls Short → Chop

```
   old daily low 3855 ─ ─ ─ ─ ─ ─
   today's low     3856  ← ONE handle short → downside continuation DIES
        ▲ deceptive double-bottom lures traders
   price rotates back to the MIDDLE of the range → odds 50/50 → NO edge
   (listless chop: short-term highs/lows swept back and forth)
```

- Falling **one handle short** (3856 vs 3855) killed the downside and flipped odds to **50/50**. See [Market Structure](../../concepts/market-structure.md).
- Most of the session offered **no clean setup** — stand aside.

---

## The Market-On-Close Reversal (3–4 PM)

```
   3:00–4:00 PM NY (pre-drawn vertical lines)
   relative equal highs ═══  ← run ABOVE = grab BUY-SIDE liquidity
        ╱╲   induces breakout longs; squeezes shorts (buy stops = willing buyers)
   ────╱  ╲  smart money SELLS into it
           ╲  MOC on-close orders drive price DOWN to the daily MIDPOINT
            ▼  breakout longs punished + shorts stopped out
```

- **Before 3:00:** run above relative equal highs for **buy-side liquidity**. See [Liquidity](../../concepts/liquidity.md).
- **MOC reversal:** sell into the liquidity → drive back to the **daily midpoint**. See [Algorithmic Programs](../../concepts/algorithmic-programs.md).

### 1-min confirmation
```
run above buy-side → swing high + FVG → retrace into gap → drive to midpoint
```
See [Fair Value Gaps](../../concepts/fair-value-gaps.md) / [Swing Points](../../concepts/swing-points.md).

---

## The Repeatable Read

```
Outside-down-close → today falls short of old low → consolidation (50/50 chop, stand aside)
→ 3–4 PM: run buy-side above equal highs → MOC reversal → back to daily midpoint
```
