## What This Episode Teaches

How to build a **daily directional bias** from the daily chart — and what to do when **consolidation** around equilibrium makes that bias unclear. The key edge introduced here is **intermarket correlation**: using NASDAQ, S&P, and Dow together, where the Dow's refusal to confirm a new low exposes a fake break.

## The Bias Framework (Step-by-Step)

### Stage 1 — Establish Daily Bias

| Step | Action | Concept |
|------|--------|---------|
| 1 | On the **daily**, read the candle closes: several down-closes after a lower low = **bearish**. | [Daily Bias](../../concepts/daily-bias.md) |
| 2 | Note if price just **broke an old low** (took [sell-side liquidity](../../concepts/liquidity.md)) then retraced. | [Liquidity](../../concepts/liquidity.md) |
| 3 | Locate price in the range: **premium / discount / equilibrium**. | [Market Structure](../../concepts/market-structure.md) |

### Stage 2 — If Consolidating, Confirm with Correlation

| Step | Action | Concept |
|------|--------|---------|
| 4 | If price hovers near **equilibrium**, bias is unclear — go to correlation. | [Daily Bias](../../concepts/daily-bias.md) |
| 5 | Compare **NASDAQ vs S&P vs Dow**. All agree → confident. | [Intermarket Correlation](../../concepts/intermarket-correlation.md) |
| 6 | **Divergence tell:** NQ & ES make a new low but **Dow does NOT** → fake break, accumulation, expect a rally. | [Intermarket Correlation](../../concepts/intermarket-correlation.md) |

### Stage 3 — Intraday Execution

| Step | Action | Concept |
|------|--------|---------|
| 7 | Drop to **15m → 5m → 2m → 1m**; mark old highs/lows as liquidity. | [Multi-Timeframe Analysis](../../concepts/multi-timeframe-analysis.md) |
| 8 | Wait for a **market structure shift** (break of swing high after a sweep) + a **fair value gap**. | [Market Structure Shift](../../concepts/market-structure-shift.md) / [Fair Value Gaps](../../concepts/fair-value-gaps.md) |
| 9 | Enter near the **order block / FVG**; stop tight just outside the liquidity zone. | [Fair Value Gaps](../../concepts/fair-value-gaps.md) |
| 10 | Use **surgical strikes** (a few handles) when bias is uncertain — don't overstay. | [Daily Bias](../../concepts/daily-bias.md) |

## Leverage by Bias Certainty

| Situation | Risk |
|-----------|------|
| Aligned with a clear bias | **Max** risk tolerance (e.g., up to ~4.5%) |
| Counter-trend | **Reduced** leverage / smaller size |
| Consolidation / unclear | Light; surgical strikes only |

See [Risk Management](../../concepts/risk-management.md).

## The Dow Divergence Tell

```
   NASDAQ:  new lower low  ✔
   S&P:     new lower low  ✔
   DOW:     NO new low     ✘   ← "lack of delivery"
                                = fake break lower → accumulation → rally
```

This is the episode's signature insight — when one correlated index **refuses to deliver** to a new extreme, the break is likely a liquidity grab. See [Intermarket Correlation](../../concepts/intermarket-correlation.md).

## Quick-Reference Checklist

```
□ Daily bias read from candle closes (down-closes = bearish)
□ Noted recent liquidity sweep (old low/high taken)
□ Price located: premium / discount / equilibrium
□ If consolidating → check NQ/ES/YM correlation
□ Divergence? (Dow not confirming) → suspect fake break
□ Intraday MSS + FVG for entry (15m→1m)
□ Leverage scaled to bias certainty (full / reduced / surgical)
□ Tight stops just outside liquidity zones
```

## The Core Insight

> A clear daily bias comes from candle closes and liquidity sweeps — but **consolidation obscures it**. When one chart is unclear, read the **correlated instruments**: the index that **refuses to make a new low/high** exposes accumulation and a fake break. Scale leverage to how certain the bias is.
