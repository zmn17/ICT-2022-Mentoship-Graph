## The Anchor

```
   Friday close ─────
        ╎ GAP  ← Sunday-night re-open (~6:00 PM local) gapped from Friday
   Sunday open ═══════  ← WEEKLY reference level (extend across the whole week)
```

Mark the Sunday-night opening gap; **extend it across the week**. FOMC/consolidation week → expect repeated returns. See [Sunday Gap Opening](../../concepts/sunday-gap-opening.md).

---

## The Delivery (down through the gap)

```
   relative equal highs ═══  early drift + small rally + SMT divergence → break down
   ┌──────┐ live FVG
   │ FVG  │  run the SELL-SIDE
   └──────┘
   Sunday open ═══════  ← trade DOWN THROUGH it (support first)
        ▼ stair-step lower via FVGs + order blocks
   3926.75 ─── near-perfect delivery (retested to the tick)
        ▲ come back UP → Sunday gap now RESISTANCE (repeated retests)
   lunch consolidation → retrace to opening range
   afternoon → cleaner setups down to 3952.75
```

- Early **SMT divergence** + broken relative equal highs → break down. See [SMT Divergence](../../concepts/smt-divergence.md).
- Live **FVG** → run **sell-side** → **through the Sunday gap**. See [Fair Value Gaps](../../concepts/fair-value-gaps.md) / [Liquidity](../../concepts/liquidity.md).
- Stair-steps lower using **FVGs + order blocks**; near-perfect delivery at **3926.75** (retested to the tick). See [Order Blocks](../../concepts/order-blocks.md).
- The **Sunday gap flips support → resistance** on the way back up; repeated retests. See [Sunday Gap Opening](../../concepts/sunday-gap-opening.md).

---

## Projecting the Next Draw

```
   unfilled HIGHER gap ┌──────┐  ← next draw (ideally to the origin candle's low)
                       │ FVG  │
                       └──────┘
   above that → daily run toward the HIGH + a 4,000 order block
```

Project the next **draw on liquidity** into the unfilled higher gap; a move above it opens a deeper daily run toward a **4,000 order block**. See [Draw on Liquidity](../../concepts/draw-on-liquidity.md).

---

## The Repeatable Read

```
Sunday-night gap = weekly anchor → extend all week
→ trade through it (FVG + OB reactions) → gap flips support↔resistance
→ project next draw into the unfilled gap beyond (toward 4,000 OB)
```
