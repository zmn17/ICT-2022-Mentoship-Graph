## What This Episode Teaches

How to apply the **same institutional order-flow concepts** — market structure, fair value gaps, liquidity, sessions — to **Forex**, using EUR/JPY as the case study. Two Forex-specific additions: **cross correlation** (read the two underlying currencies to set bias) and **Fibonacci / standard-deviation targeting** (when the intraday chart has no clean range).

---

## Learn the Concepts (Read in This Order)

```
1. Intermarket Correlation → Read EUR (6E) + Yen to bias the cross
2. Market Structure Shift  → Validate with breaks of swing highs/lows
3. Liquidity               → Old highs/lows + relative equal lows
4. Fair Value Gaps         → Entry zones on the retrace
5. Trading Sessions        → London hunt (2–5am), NY execute (7–10am)
6. Fibonacci Targeting     → Std-dev projections when no range exists
```

## The Entry Model (Step-by-Step)

### Stage 1 — Set Bias via Correlation

| Step | Action | Concept |
|------|--------|---------|
| 1 | Don't read EUR/JPY alone — check the **two currencies**. | [Intermarket Correlation](../../concepts/intermarket-correlation.md) |
| 2 | **Euro (6E futures)** in an uptrend = Euro strength. | [Intermarket Correlation](../../concepts/intermarket-correlation.md) |
| 3 | **Yen** weak/consolidating → **bullish bias on EUR/JPY**. | [Intermarket Correlation](../../concepts/intermarket-correlation.md) |

### Stage 2 — Validate Structure (Daily → Intraday)

| Step | Action | Concept |
|------|--------|---------|
| 4 | On the **daily**, mark **relative equal lows** and confirm price **breaks above swing highs** (bullish structure). | [Market Structure Shift](../../concepts/market-structure-shift.md) |
| 5 | Note **fair value gaps** between prior legs — price retraces to fill them. | [Fair Value Gaps](../../concepts/fair-value-gaps.md) |
| 6 | Liquidity is **drawn to old highs** (buy stops) even if not immediately reached. | [Liquidity](../../concepts/liquidity.md) |

### Stage 3 — Time the Entry by Session

| Step | Action | Concept |
|------|--------|---------|
| 7 | **London (2–5 AM NY):** hunt for FVG setups. | [Trading Sessions](../../concepts/trading-sessions.md) |
| 8 | **New York (7–10 AM NY):** execute on the identified FVG. | [Trading Sessions](../../concepts/trading-sessions.md) |
| 9 | Refine on **15m → 5m → 3m → 1m**; enter on the retrace into the FVG. | [Fair Value Gaps](../../concepts/fair-value-gaps.md) |

### Stage 4 — Target with Fibonacci

| Step | Action | Concept |
|------|--------|---------|
| 10 | No clean range? Anchor a **Fib between the post-consolidation swing low and high**. | [Fibonacci Targeting](../../concepts/fibonacci-targeting.md) |
| 11 | Target the **standard-deviation projection** (e.g., std dev −1 ≈ **133.15**). | [Fibonacci Targeting](../../concepts/fibonacci-targeting.md) |

## Quick-Reference Checklist

```
□ Bias set via correlation (EUR strong via 6E + Yen weak → bullish EUR/JPY)
□ Daily structure validated (break above swing highs; relative equal lows marked)
□ Fair value gaps between legs identified
□ Session timing: London hunt / New York execute
□ Refined entry on 15m→1m into the FVG
□ Target set via Fib std-dev projection (e.g., ~133.15) when no range exists
```

## The Core Insight

> The order-flow model works on **Forex** too. For a cross, set bias by reading the **two underlying currencies** (Euro strong + Yen weak = long EUR/JPY), enter on a **fair value gap** during the London/NY sessions, and — when there's no clean range — project a target with **Fibonacci standard deviations**.
