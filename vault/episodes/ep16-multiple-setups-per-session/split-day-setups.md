## Daily Context

```
   ═══════════ relative equal highs (buy-side)  ← ran above, printed swing high
        ╲
   premium (stretched up) → bias DOWN
        │
   ── down-close daily BULLISH ORDER BLOCK ~4501–4504 ──  ← the DRAW (discount)
```

Price ran above relative equal highs, printed a swing high, and left a **down-close daily bullish order block** below (~4501–4504) — the **draw on liquidity** for a bearish morning. See [Draw on Liquidity](../../concepts/draw-on-liquidity.md).

---

## Morning Session (Bearish, 8:30–noon)

```
   Judas swing above midnight/8:30 opens (manipulation)
        ╱╲
   ────╱  ╲  displacement LOWER (distribution)
           ╲  sweeps sell-side under relative equal lows
   ┌───┐    ╲
   │FVG│◄─┐  ╲___ into daily OB ~4501–4504 (target)
   └───┘  │      body-anchored Fib (4514→4501) lands on day's low
   short retrace into each FVG
```

1. **Judas swing** above the midnight & 8:30 opens = manipulation. See [Judas Swing](../../concepts/judas-swing.md) / [Opening Price](../../concepts/opening-price.md).
2. **Displacement lower** (distribution) sweeps sell-side liquidity and drives into the daily OB.
3. **Target** precisely via a **body-anchored Fibonacci** (anchor open/close, never wicks): 4514 → 4501 lands on the day's low. See [Fibonacci Targeting](../../concepts/fibonacci-targeting.md).
4. Down through **15 → 5 → 1-min**, the move fragments into **several shorts**, each: break a short-term low (displacement) → FVG left behind → retrace into FVG → short → next liquidity pool. See [Market Structure Shift](../../concepts/market-structure-shift.md).

---

## PM Session (Bullish, after 1:00)

```
   buy stops ═══════════  (premium target overhead)
        ▲ reprice discount → premium
        │
   ┌────┴─────┐ mitigation block + imbalance  ← PM LONG entry (discount)
   │  MB+FVG  │
   └──────────┘
   relative equal lows: retail sells broken support → smart money buys it
```

1. After the daily objective is met, the afternoon becomes a **buy-side liquidity pool**. See [Draw on Liquidity](../../concepts/draw-on-liquidity.md).
2. Smart money **engineers retail sell flow** — traders sell the return to broken support at relative equal lows — and **buys against it**. See [Liquidity](../../concepts/liquidity.md).
3. **PM long:** enter at a **mitigation block + imbalance** in discount. See [Mitigation Block](../../concepts/mitigation-block.md).
4. **PM short** (alternate): market structure shift + FVG.
5. Reprice **discount → premium** to fill the buy stops overhead. See [Market Structure](../../concepts/market-structure.md).

---

## The Repeating Short (Morning, any timeframe)

```
break short-term low (displacement) → FVG left behind
→ price retraces into FVG → SHORT → target next liquidity pool
```

This is the same fractal setup 2–3× in the morning. Missing one is fine — it **repeats**.
