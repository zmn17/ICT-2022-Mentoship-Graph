## The Entry Model (Recap of Ep 3, Applied)

| Step | Action | Concept |
|------|--------|---------|
| 1 | Define the **low-to-high dealing range**; drop a **Fib**; mark **50% equilibrium**. | [Market Structure](../../concepts/market-structure.md) |
| 2 | Run-up leaves price in **premium** (expensive) → bias to **short**. | [Market Structure](../../concepts/market-structure.md) |
| 3 | Wait for a **run above an old high** (liquidity) — the **8:30 open** starts the hunt. | [Liquidity](../../concepts/liquidity.md) / [Trading Sessions](../../concepts/trading-sessions.md) |
| 4 | Require a **break of the swing low** = bearish **market structure shift**. | [Market Structure Shift](../../concepts/market-structure-shift.md) |
| 5 | The break must leave a **fair value gap**. **No FVG = no trade.** | [Fair Value Gaps](../../concepts/fair-value-gaps.md) |
| 6 | **Short** the retrace back **into the FVG**. | [Fair Value Gaps](../../concepts/fair-value-gaps.md) |
| 7 | **Target** a discount old low / swing low / imbalance below equilibrium. | [Fibonacci Targeting](../../concepts/fibonacci-targeting.md) |

## The Discipline Lesson (Why the Counter-Example Matters)

> "You have to wait for the setup — you can't force it, you can't think it's there. It's there or it's not."

The Nasdaq's first run took relative equal highs but **did not create a fair value gap** → it failed the criteria → **no trade**. Only a later break that *did* leave a gap qualified. The rules force the market to **show its hand**. See [Trading Psychology](../../concepts/trading-psychology.md).

## Quick-Reference Checklist

```
□ Low-to-high range defined; Fib dropped; equilibrium marked
□ Price in premium (post run-up) → short bias
□ Run above an old high / relative equal highs (liquidity)
□ 8:30 hunt window
□ Break of swing low = bearish MSS
□ FVG left by the break (NO FVG = NO TRADE)
□ Short the retrace into the FVG
□ Target discount low / imbalance below equilibrium
□ Journaled: shift→gap time, entry→target time, heat endured
```

## The Core Insight

> This is the [Episode 3](../ep03-internal-range-liquidity-mss/README.md) model in practice: range → Fib → premium → run the old high → MSS → **FVG** → short into the gap → discount target. The counter-example proves the rules are binary — **no fair value gap, no trade**. Journal every instance and train your eye; setups recur with similarities, never exact copies.
