## What This Teaches

A live application of **premium/discount** across hourly → 5-min → 1-min. Define the **dealing range** (swing low → swing high), drop a **Fib** to mark **equilibrium (50%)**: above = **premium**, below = **discount**. Price into a **discount fair value gap** = **oversold with no indicator**. The lower timeframes then show liquidity engineered: **sell-side swept** below relative equal lows at 9:30, a **market structure shift**, **rebalance into a 5-min FVG**, then a rally into **deep premium** to reach the **morning high** (the draw).

## Reading the Stop Runs

> Sell stops below the lows **are the liquidity** ICT buys into; retail longs get suckered in near the bottom and knocked out. The **aggressive down move with no clean model entry** is itself the **tell** — price was merely **rushing to clear liquidity** before reversing (the algorithm tipping its hand). See [Institutional Order Flow](../../concepts/institutional-order-flow.md) / [Tape Reading](../../concepts/tape-reading.md).

## The Core Insight

> **Premium/discount needs no indicator** — just a **dealing range** and its **equilibrium**. Price in a **discount FVG** is oversold; expect delivery to **premium**. The **9:30 sell-side sweep** (a stop run with no clean model entry = the tell) sets the **shift in structure**, and price **rebalances a FVG** before rallying to the **morning high** — the draw on liquidity.
