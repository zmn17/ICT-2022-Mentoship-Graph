## What This Teaches

A **live tape-reading** exercise: a **counter-bias long** taken *because* sell-side liquidity was swept below relative equal lows and price traded into a **fair value gap near 4000**. ICT narrates the whole read in real time — mapping buy/sell-side pools, defining the draw (buy stops above the **4051.25** HOD), **pyramiding** into bullish order blocks, managing the stop beneath structure, and scaling out at the buy-side objective. Booked **~$5,000 hypothetical**.

---

## Learn the Concepts (Read in This Order)

```
1. Institutional Order Flow → Read where price will be delivered, not chart patterns
2. Tape Reading          → Call the move in advance, "no waffle"
3. Draw on Liquidity     → Buy stops above the 4051.25 HOD = the objective
4. Liquidity             → Sell-side below equal lows was taken (the fuel)
5. Fair Value Gaps       → Long into the FVG near 4000
6. Order Blocks          → Pyramid into bullish OBs (down-close candles)
7. Daily Bias            → Counter-bias long is evidence-based, not blind
8. Risk Management       → Pyramid 4→6; stop beneath structure; scale out
```

## The Trade (Step-by-Step)

| Step | Action | Concept |
|------|--------|---------|
| 1 | **Counter-bias long** — justified: market dropped 3× and took yesterday's low with **no follow-through**. | [Daily Bias](../../concepts/daily-bias.md) |
| 2 | **Sell-side** below relative equal lows was **taken** (bought) → the fuel. | [Liquidity](../../concepts/liquidity.md) |
| 3 | Price traded into a **fair value gap near 4000** = the long entry. | [Fair Value Gaps](../../concepts/fair-value-gaps.md) |
| 4 | **Draw:** buy stops above the **4051.25** high of day (best case; doesn't need to reach it to profit). | [Draw on Liquidity](../../concepts/draw-on-liquidity.md) |
| 5 | Retail reads it as a **bear flag** / sells the rebalanced 8:30 imbalance — that's the **liquidity being targeted**. | [Institutional Order Flow](../../concepts/institutional-order-flow.md) |

| 8 | **Scale out 4 contracts** above buy-side liquidity with a market order; let the rest deliver. | [Risk Management](../../concepts/risk-management.md) |
| 9 | **Ignore the P&L, watch price.** Result: **~$5,000 hypothetical**, called in advance. | [Tape Reading](../../concepts/tape-reading.md) |

## Counter-Bias, With Evidence

> Trading **against** the daily bias is a **deliberate exception**, taken only with supporting evidence:
> - sell-side **swept** (liquidity taken)
> - price inside a **fair value gap**
> - downside momentum **stalled** (3 drops, no follow-through)
>
> It is **not** fighting the trend blindly.

---

## Quick-Reference Checklist

```
□ Sell-side swept below relative equal lows (fuel present)
□ Price into a fair value gap (~4000) = long entry
□ Draw defined = buy stops above HOD (4051.25)
□ Counter-bias justified by stalled downside (evidence, not blind)
□ Pyramid into bullish order blocks (4→6); annotate in real time
□ Stop beneath structure (inside lower FVG / below old STH)
□ Scale out above buy-side; watch price not P&L
```

## The Core Insight

> Tape reading is **calling the move before it happens** — no waffle. Sell-side was taken, price sits in a **fair value gap**, downside stalled, so the next delivery is **up into buy-side liquidity**. The retail "bear flag" sellers **are** the liquidity. **Pyramid** into bullish order blocks, manage the stop beneath structure, and scale out at the buy-side draw.
