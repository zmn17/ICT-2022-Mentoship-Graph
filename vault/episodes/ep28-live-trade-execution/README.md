## The Live-Execution Template (Reference)

```
1. Bias + draw on liquidity  → know the objective (see Draw on Liquidity)
2. Wait for the sequence     → liquidity raid → DISPLACEMENT → FVG (see Ep 24)
3. Enter at OTE / the FVG    → tight structural stop (see Optimal Trade Entry)
4. Manage                    → scale out, roll to break-even, watch price not P&L
5. Exit at the draw / objective
```
