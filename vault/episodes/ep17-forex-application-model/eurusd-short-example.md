## Daily — Bias & Draw

```
   ═══════════ relative equal highs  ← ran above = buy-side SWEPT
        ╲
        ╲  roll over toward what's resting below
   ─ ─ ─ ╲─ ─ relative equal lows ("two little devils") = sell-side (unswept)
          ▼
   1.0900 ══════  BIG FIGURE magnet (liquidity + Fib converge)
```

- Buy-side above the equal highs **taken**; **Monday's low not yet swept** = **unfinished business** below → **bearish** next daily candle. See [Draw on Liquidity](../../concepts/draw-on-liquidity.md).
- Downside magnet = the **1.09 big figure**; mark the **00/20/50/80** levels. See [Institutional Price Levels](../../concepts/institutional-price-levels.md).

## Intraday — Power of 3 in the NY Open Kill Zone

```
   after midnight NY: PROTRACT higher (Judas swing) into hourly imbalance = daily high
        ╱╲  ← above midnight open = smart money accumulates SHORTS
   ────╱  ╲  bearish order block (~7:00) + FVG
           ╲  MSS + lower high
            ▼ distribute lower (NY Open Kill Zone 7–10 AM)
```

1. After **midnight NY**, price **protracts higher** — a [Judas swing](../../concepts/judas-swing.md) into an hourly imbalance forming the daily high.
2. **Above the midnight open** = short-accumulation zone. See [Opening Price](../../concepts/opening-price.md).
3. Hunt in the **NY Open Kill Zone (7–10 AM)**. See [Trading Sessions](../../concepts/trading-sessions.md).
4. Price tags a **bearish order block** + **FVG** (~7:00) → **market structure shift** + lower high → descent. See [Order Blocks](../../concepts/order-blocks.md) / [Fair Value Gaps](../../concepts/fair-value-gaps.md) / [Market Structure Shift](../../concepts/market-structure-shift.md).

## The 10:00 News Wildcard

```
   10:00 ISM Services PMI ─┐
                           ▼ retrace UP into the order block
                              (shakes out / stops trailed traders)
                           ▼ then algorithm closes a small FVG
                           ▼ aggressively runs the short-term low
   1.09 ═══════════  target reached (Fib extension + equal lows)
```

The high-impact **10:00 ISM Services PMI** is the wildcard that retraces into the OB and shakes traders out — *before* the run down to ~1.09. See [Economic Calendar](../../concepts/economic-calendar.md).

## The Repeatable Sequence (Forex)

```
Daily: buy-side taken + sell-side unfinished → bearish; magnet = 1.09 big figure
→ Fib std-dev targets → midnight open short-accumulation
→ NY Open Kill Zone: Judas swing up → bearish OB + FVG → MSS
→ news wildcard shakes stops → run to 1.09
```
