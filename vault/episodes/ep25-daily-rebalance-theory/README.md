## What This Teaches

**Daily Rebalance Theory:** a large-range distribution day (Monday) leaves a **daily inefficiency**; the next day **retraces to rebalance** it — up to a **logical level (Friday's low)** — **before resuming lower**. That rally is a **Judas swing to premium** that tricks traders into thinking a low is in. The bias is bearish (May seasonal + run above equal highs), so after the rebalance a **FVG + MSS + displacement** delivers price to discount sell-side liquidity.

## The Rebalance Trap

```
   Friday low ═════  ← rebalance level (premium) — "low is in?" NO, it's a Judas swing
        ▲ Tuesday rallies UP to rebalance…
   ─────┼─────  Monday = large distribution day (daily inefficiency)
        ▼ …then FVG + MSS + displacement DOWN → discount sell-side
```

---

## Quick-Reference Checklist

```
□ Big-range daily move noted (the inefficiency) + bearish bias (seasonal/OB)
□ Next day: buy-side run first at 9:30 (confirms bias)
□ Rally rebalances the prior big day to a logical level (prior day low/high)
□ Recognize it as a Judas swing to PREMIUM (not a reversal)
□ FVG + MSS + displacement at the rebalance level → short
□ Target discount sell-side; offset/limits during lunch
```

## The Core Insight

> A big **daily** move leaves a **daily inefficiency**. The next day **retraces to rebalance** it to a logical level — a move that **looks bullish but is a Judas swing to premium**. Judge the imbalance on the **daily** (not the 15-min), and once the rebalance completes, the **FVG + MSS + displacement** resumes the bias toward the **discount** draw.
