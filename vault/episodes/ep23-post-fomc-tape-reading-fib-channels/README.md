## What This Teaches

A **live post-FOMC tape read**: wait out the FOMC shakeout, identify an **untraded higher-timeframe FVG** and a **bearish order block** as the "end of the road" for the drop, forecast the drop into that zone **then** a run through Tuesday's highs — narrating each leg **in advance**. Introduces the **sweep vs. run** distinction and a **Fibonacci expansion** target (wicks vs. bodies keyed to volatility).

## The Core Insight

> On FOMC, **patience first** — wait out the staged shakeout. Then **read the tape in advance**: an untraded **HTF fair value gap** + a **bearish order block** mark the end of the drop; price dips into the zone, turns, and **runs** the prior highs. Distinguish a **sweep** (fade) from a **run** (follow), and project the upside with a **Fib expansion** anchored to the **wicks** (FOMC uses the extremes).
